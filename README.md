# BandApp

BandApp — это платформа для музыкального сообщества, которая упрощает организацию репетиций, взаимодействие между участниками групп и продвижение музыкальных мероприятий.

Основная цель проекта — сократить время, которое музыканты тратят на организационные задачи, и позволить им сосредоточиться на творчестве.

---

## Основные возможности

BandApp объединяет несколько ключевых модулей:

- управление музыкальной группой
- поиск и бронирование репетиционных студий
- поиск музыкантов для группы
- афиша мероприятий
- инструменты для работы с музыкальными материалами (табулатуры, тексты)

---

## Архитектура системы

Система состоит из нескольких основных компонентов.

**Frontend**

Кроссплатформенное мобильное приложение, разработанное с использованием React Native.

**Backend**

REST API, отвечающий за бизнес-логику, аутентификацию пользователей и обработку данных.

**База данных**

Реляционная база данных, в которой хранятся данные о пользователях, группах, репетиционных студиях, бронированиях и событиях.

---

## Внешние интеграции

В системе предусмотрены интеграции со сторонними сервисами:

- Spotify / Apple Music — получение информации о релизах и артистах
- Songsterr — табулатуры
- ЮKassa — онлайн-платежи
- картографические API — отображение студий и мероприятий на карте

## 📦 Быстрый старт

### Требования
- Git
- Node.js (версии 18 или выше)
- npm или yarn
- Docker (для запуска базы данных)
- PostgreSQL (если запуск без Docker)

### Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/bandapp/bandapp.git
cd bandapp

# Запустить инфраструктуру (БД, Redis и т.д.)
docker-compose up -d

# Установить зависимости бэкенда
cd backend && npm install

# Запустить бэкенд в режиме разработки
npm run dev

# В другом терминале — мобильное приложение
cd mobile && npm install
npm run ios   # или android
